# 前端设计系统规范

> 本文档定义统一的视觉语言、设计 Token、颜色/字体/间距规范，确保所有页面风格一致，提升开发效率与用户体验。

---

## 1. 设计系统概览

设计系统由 **Design Tokens**、**组件库**、**主题配置** 三大层组成，形成 **原子 → 分子 → 组织 → 页面** 的层级结构。

- **Design Tokens**：颜色、字体、间距、阴影等基础变量，统一存放在 `theme/` 中，使用 CSS 变量 + Ant Design Token 覆盖实现。
- **组件库**：基于 Ant Design 二次封装的业务组件（Button、Table、Form 等），所有样式通过 Token 渲染。
- **主题配置**：支持亮色、暗色两套主题，可通过全局切换实现。

---

## 2. 颜色系统

| 颜色类型 | 变量名 | 示例值 | 用途 |
|---|---|---|---|
| 主色 | `--color-primary` | `hsl(220, 94%, 56%)` | 交互按钮、链接、强调色 |
| 成功 | `--color-success` | `hsl(150, 70%, 48%)` | 成功提示、状态 |
| 警告 | `--color-warning` | `hsl(45, 100%, 51%)` | 警告提示 |
| 错误 | `--color-danger` | `hsl(0, 78%, 55%)` | 错误提示 |
| 中性 | `--color-neutral-1~9` | `hsl(0,0%,5%)` → `hsl(0,0%,95%)` | 背景、文字、分割线 |

**渐变**：`--gradient-primary` = `linear-gradient(90deg, var(--color-primary), var(--color-primary-hover))`

---

## 3. 排版系统

| 令牌 | 变量名 | 示例值 | 说明 |
|---|---|---|---|
| 基础字体 | `--font-family-base` | `"Inter", system-ui, sans-serif` | 全局默认字体 |
| 标题系列 | `--font-size-h1`~`--font-size-h6` | `2.5rem` → `1rem` | 标题大小递减比例 1.25 |
| 正文字体 | `--font-size-base` | `1rem` | 正文、段落 |
| 行高 | `--line-height-base` | `1.5` | 基础行高 |
| 字重 | `--font-weight-regular` / `--font-weight-bold` | `400` / `600` | 常规与强调 |

## 4. 间距系统（Spacing Scale）

采用 **4px 基准**，通过指数级递增构建间距 Token。

| Token | 变量名 | 像素值 |
|---|---|---|
| `--spacing-0` | `0` | `0px`
| `--spacing-1` | `4` | `4px`
| `--spacing-2` | `8` | `8px`
| `--spacing-3` | `12` | `12px`
| `--spacing-4` | `16` | `16px`
| `--spacing-5` | `24` | `24px`
| `--spacing-6` | `32` | `32px`
| `--spacing-7` | `48` | `48px`
| `--spacing-8` | `64` | `64px`

**使用方式**：在 Tailwind 配置中映射为 `spacing`，在 CSS 中直接使用 `var(--spacing-4)`。

## 5. 组件库规范

1. **原子组件**（Button、Input、Icon 等）
   - 仅接受 **props**，不涉及业务逻辑。
   - 样式使用 `className` 绑定 Design Token（如 `bg-primary`）。
2. **业务组件**（UserTable、RoleSelect）
   - 组合原子组件，封装交互和数据映射。
   - 通过 `useQuery`/`useMutation` 处理数据，不直接在 UI 层写请求。
3. **组件命名**：采用 PascalCase，文件同名 `UserTable.tsx`。
4. **文档化**：使用 Storybook 自动生成组件预览与交互文档。

---

## 6. 暗黑主题实现

- 在 `theme/dark-token.ts` 中覆盖 `--color-primary`、`--color-neutral` 等变量。
- 使用 CSS `prefers-color-scheme` media 查询或全局 Store `useThemeStore` 切换 `data-theme="dark"`。
- 所有组件通过 CSS 变量自动适配，无需额外代码。

## 7. 设计系统维护流程

| 步骤 | 负责人 | 产出 |
|---|---|---|
| 需求提出 | 产品 / 设计 | 新 Token 或组件需求 |
| 设计评审 | 设计 + 前端 | 设计稿 + 交互原型 |
| 实现 & Review | 前端 | 完成 Token/组件，提交 PR |
| 验收 | QA | UI 对齐、单元/视觉测试 |
| 发布 | CI | 自动发布至 npm 私仓并更新 Storybook |
