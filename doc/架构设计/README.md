## 阅读建议
- 想快速知道前端怎么搭时，先看《前端技术栈总览》。
- 落地工程结构与模块拆分时，再看《前端工程结构设计》。
- 涉及菜单、动态路由、状态边界时，分别看路由与状态管理文档。

## 维护规则
- 本目录只维护前端结构、边界和长期约定，不承载短期页面任务清单。
- 版本、包管理器和本地启动方式变化时，应同步更新环境搭建文档并对齐版本矩阵。
- 鉴权链路与安全基线相关主题应回链到安全目录的 SSOT 文档。
