# 项目技术栈与结构规范

## 技术栈

### 前端框架与构建工具
*   **[React](https://react.dev) (v19)**: 项目的核心 UI 框架。
*   **[Vite](https://vitejs.dev)**: 构建工具，提供快速的开发服务器和优化的构建输出。
*   **[TypeScript](https://www.typescriptlang.org)**: 主要开发语言，为 JavaScript 添加静态类型检查。

### UI 与样式
*   **[Tailwind CSS](https://tailwindcss.com)**: 原子类优先的 CSS 框架，用于快速构建自定义用户界面。
*   **[Vanilla-extract](https://vanilla-extract.style)**: 使用 TypeScript 编写零运行时、类型安全的 CSS 样式。
*   **[Framer Motion](https://www.framer.com/motion)**: 用于在 React 中创建流畅动画的强大库。
*   **[shadcn/ui](https://ui.shadcn.com)**: 基于 Radix UI 和 Tailwind CSS 构建的一套可复用组件。
*   **[Iconify](https://iconify.design)**: 在项目中使用各种图标集。

### 状态管理
*   **[Zustand](https://zustand-demo.pmnd.rs)**: 轻量级且快速的全州状态管理库。
*   **[TanStack Query (React Query)](https://tanstack.com/query/latest)**: 用于管理服务器状态，处理数据获取、缓存和同步。

### 路由
*   **[React Router](https://reactrouter.com)**: 处理应用内导航和路由。

### 表单处理
*   **[React Hook Form](https://react-hook-form.com)**: 用于构建高性能、灵活且易于使用的表单。
*   **[Zod](https://zod.dev)**: TypeScript 优先的模式声明和验证库，常与 React Hook Form 配合使用。

### 数据可视化
*   **[ApexCharts](https://apexcharts.com)**: 用于创建交互式图表和数据可视化。

### 国际化
*   **[i18next](https://www.i18next.com)**: 强大的国际化框架，支持多语言。

### 开发工具与代码质量
*   **[Biome](https://biomejs.dev)**: 高性能的代码格式化和检查工具，确保一致的代码风格和质量。
*   **[Lefthook](https://github.com/evilmartians/lefthook)**: Git 钩子管理器，在提交前自动运行脚本（如 lint、测试）。
*   **[Mock Service Worker (MSW)](https://mswjs.io)**: 在开发环境中拦截请求并返回 Mock 数据，实现纯前端开发和测试。
*   **[pnpm](https://pnpm.io)**: 项目的包管理器。

## 项目结构
```
├─ public/
│  └─ mockServiceWorker.js      # MSW 启动脚本（自动生成的 Service Worker 脚本）
├─ src/                         # 核心源码
│  ├─ _mock/                    # Mock API 数据和处理程序
│  ├─ api/                      # API 请求服务
│  ├─ assets/                   # 静态资源（图片、图标）
│  ├─ components/               # 全局可复用组件
│  ├─ hooks/                    # 自定义 React Hooks
│  ├─ layouts/                  # 页面布局组件
│  ├─ locales/                  # 国际化 (i18n) 语言包
│  ├─ pages/                    # 页面组件（对应路由）
│  ├─ routes/                   # 路由配置
│  ├─ store/                    # 全局状态管理 (Zustand)
│  ├─ theme/                    # 主题和样式配置
│  ├─ types/                    # 全局 TypeScript 类型定义
│  ├─ ui/                       # 基础 UI 组件 (shadcn/ui)
│  ├─ utils/                    # 工具函数
│  ├─ App.tsx                   # 根应用组件
│  └─ main.tsx                  # 应用入口点
├─ .env                         # 环境变量文件
├─ .env.development             # 开发环境变量
├─ .env.production              # 生产环境变量
├─ package.json                 # 项目依赖和脚本
├─ vite.config.ts               # Vite 配置文件
├─ tailwind.config.ts           # Tailwind CSS 配置文件
├─ tsconfig.json                # TypeScript 配置文件
└─ biome.json                   # Biome 格式化/检查配置文件
```
